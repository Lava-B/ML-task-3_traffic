
# ML Task 3 - Traffic Volume Prediction

Comparing 3 learning models:
Random Forest, AdaBoost, XGBoost


## Results

Random Forest:
MAE = 203.18
RMSE = 375.55
R² = 0.9643

AdaBoost:
MAE = 630.76
RMSE = 840.42
R² = 0.8213

XGBoost:
MAE = 228.41
RMSE = 384.34
R² = 0.9626


Random Forest performed best, with XGBoost performing almost the same. AdaBoost performed worse than the other two models.



I learned ensemble learning and combining multiple models. Random Forest uses decision trees and combines predictions. AdaBoost corrects mistakes made by previous models. XGBoost is a boosting algorithm that improves by reducing errors.
